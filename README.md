# UvA DevOps Software QA LLM MVP

A Streamlit app for uploading and reading files.

## Requirements

- Python 3.11+
- pip

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
streamlit run streamlit_app.py
```

The app will be available at `http://localhost:8501`.
